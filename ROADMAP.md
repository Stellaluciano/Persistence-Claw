# Roadmap

## Near-term next steps

1. Add native Node bindings for Rust diff engine (N-API) to reduce process-spawn overhead.
2. Add SSIM/feature-based diff plugins for more robust UI change detection.
3. Introduce richer scene graph memory with element identity tracking.
4. Add real desktop capture providers (Wayland/X11/macOS/Windows abstractions).
5. Add OpenClaw adapter package for live runtime integration tests.

## Research and benchmarking extensions

- Run longer synthetic workloads and report confidence intervals.
- Add task-success fidelity metrics alongside token reduction.
- Evaluate adaptive thresholds learned from task traces.
