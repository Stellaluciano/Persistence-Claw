# Contributing

Thanks for your interest in Persistence Claw.

- Use `pnpm format && pnpm lint && pnpm test` before opening a PR.
- Keep module boundaries clean (`capture`, `memory`, `router`, `core`, `sdk`).
- Favor deterministic fixtures for reproducible benchmark behavior.

**Rights notice:** Contributions are accepted under an all-rights-reserved project policy unless explicitly agreed otherwise by maintainers.
